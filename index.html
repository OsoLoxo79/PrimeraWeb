aprendi muchas etiquetas y sus atributos, obvio aun no las manejo al 100% pero van siendo utiles.

En HTML5 hay que ser muy cuidadoso a la hora de ordenar la pagina, y esto lo hacemos para que sea mas facil de encontrar para el buscador y para que utilizando conocimientos de CSS se facilite la "decoracion" de la pag.

Dentro del body de la pagina esto es el encabezado delimitado por la etiqueta "header"

A partir de esta seccion nos encontramos en la etiqueta "main"

Precisamente nos encontramos en la etiqueta "section" que ayuda a ramificar toda la informacion que pondriamos en el "main" para lograr esto debimos pasar hasta ahora por las etiquetas:

  • body: que es la principal, donde esta tooodo el contendo de esta web
  • header: donde encerramos el encabezado "h1" y varios parrafos "p"
  • Coloque una etiqueta llamada "hr" que es la que crea esa linea que divide las secciones.
  • llegamos a "main" donde puse la etiqueta "section en la cual estoy creando esta lista por medio de la etiqueta "ul" y cada uno de estos puntos es un "li"

luego de el "main" (contenido principal) viene el "footer" que es donde se pone toda la informacion de contacto entre varias cosas interesantes.

Links

Estos se crean siguiendo estos pasos:

  1. dentro de la seccion que nos interese colocar el link se pone la etiqueta "a"
  2. Si la intencion es redirigir al lector a una pagina de nuestro dominio se le agrega a la etiqueta el atributo "href="(nombre del .html creado para redirigir)"
  3. se le sigue el cierre del nombre del coc.html con ">" a continuacion se le escribe el nombre que le vamos a dar al link.
  4. cerramos la etiqueta "a"

y ya ta.
