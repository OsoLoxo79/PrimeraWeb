Los formularios:

estos siempre van a estar dentro de la etiqueta "form" por que de otra forma el lector html no lo reconoce como tal.

Los formularios son entendidos por el navegador a traves de la etiqueta "input" y sus atributos.

  1. Dentro de la etiqueta "main" va la etiqueta "form"
  2. Dentro de la etiqueta "form" va el famoso "input" que es donde el usuario o lector colocara los datos que nosotros estamos interesados en obtener

La etiqueta "input" tiene muchos atributos, los cuales deben ser utilizados para que el usuario o lector sepa que dato debe colocar en el campo

El atributo "type" es el que va a dar forma al campo a traves de el tipo de campor que queramos crear.
algunos de los tipos son:

  • text: Deja al usuario escribir en el campo
  • number: El usuario solo puede colocar numeros en el campo
  • password: permite al usuario colocar palabras, letras o simbolos, devolviendo una escritura en puntos para su privacidad
  • Entre muchos otros.

    • Dentro de la etiqueta "imput" ademas del atributo "type" encontramos muchos otros interesantes como:

    • placeholder: Atributo que nos deja escribir que tipo de dato debe ingresar el usuario.
    • name: Este nos deja hacerle entender al buscador como se llama el espacio en el que el usuario esta rellenando datos.

Debo hacer una mencion de otra etiqueta llamada "label" que nos permite escribir delante del campo el tipo de informacion que el usuario debe escribir, aunque es facilmente remplazable con el "placeholder"
esta etiqueta usa el atributo "for" que hace referencia al "name" de la etiqueta "input"

Uso de "input" y "label":

input: inmpt type="text" name="" placeholder"">

label: label for="(name del inmput)">(texto descriptivo del dato a rellenar) /label>

de esta forma logramos recaudar datos
